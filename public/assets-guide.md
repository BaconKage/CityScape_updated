# Asset folders

Drop files into these folders directly:

- Partner logos: `public/partner-logos/`
  - Use file names that match your entries in `src/data/siteData.js` (for example: `millionaire-tower.png`)

- Project/hero media: `public/images/projects/` and `public/images/hero/`

- Founder/team photos: `public/images/team/`

- General UI/background graphics (if needed): `public/images/ui/`

- Branding assets (logo, mark, favicons if any): `public/images/brand/`

All paths are served from `/` in the app, e.g. `/partner-logos/your-logo.png`.
